# Bot Ladder 1-15 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Mevcut Kolay/Orta/Zor ve karakter yapısını bozmadan, chess.com tarzı 1-15 bot rakip sistemi eklemek.

**Architecture:** Botlar mevcut AI motorunun üstünde çalışan ekstra bir seçim katmanı olacak. Klasik modda bugünkü `difficulty + aiPersonaId` aynen kalacak; bot seçilirse `botId` üzerinden zorluk, yıldız, karakter, açılış kitabı önceliği ve ufak AI stil farkları uygulanacak.

**Tech Stack:** Vanilla JS, Vite, mevcut `GameState`, `AIProfiles`, `AIPersonas`, `OpeningBook`, `main.js`, Node test runner.

---

## Tasarım Kararı

Mevcut oyun yapısı korunacak:

- Klasik seçim: `Kolay / Orta / Zor + Timur/Beyazıd/Uluğ Bey/Saray Veziri`
- Yeni seçim: `Botlar 1-15`
- Bot seçimi opsiyonel olacak.
- Bot seçilmezse oyun bugünkü gibi çalışacak.
- Bot seçilirse botun kendi `difficulty`, `personaId`, `openingBookPreferences`, `styleModifiers` ayarları uygulanacak.
- 1-15 botlar 1-5 yıldız arası gruplanacak.
- Her yıldızda 3 bot olacak.

Add helper:

```js
function applyBotModifiers(profile, botId) {
  if (!isAIBotId(botId)) return profile;

  const bot = getAIBot(botId);
  const baseProfile = AI_PROFILES[bot.difficulty] || profile;
  const personaProfile = applyPersonaModifiers(baseProfile, bot.personaId);
  const decisionStyle = { ...personaProfile.decisionStyle };

  Object.entries(bot.engineModifiers || {}).forEach(([key, value]) => {
    if (Number.isFinite(decisionStyle[key]) && Number.isFinite(value)) {
      decisionStyle[key] = clampNumber(decisionStyle[key] * value, 0, 2);
    }
  });

  return Object.freeze({
    ...personaProfile,
    id: `${personaProfile.id}:${bot.id}`,
    botId: bot.id,
    botLevel: bot.level,
    botStars: bot.stars,
    botLabel: bot.labels?.tr || bot.id,
    openingBookPreferences: Object.freeze([...(bot.openingBookPreferences || [])]),
    decisionStyle: Object.freeze(decisionStyle)
  });
}
```

Update export:

```js
export function getAIProfile(difficulty = 'medium', personaId = null, botId = null) {
  const profile = AI_PROFILES[difficulty] || AI_PROFILES.medium;
  if (isAIBotId(botId)) return applyBotModifiers(profile, botId);
  return personaId ? applyPersonaModifiers(profile, personaId) : profile;
}
```
